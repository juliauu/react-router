# Kotlin Web Site Migration

This repository contains a migrated version of the Kotlin website homepage, converted to **React Router 7 Framework Mode** with **Server-Side Rendering (SSR)**.

## Running

### Install dependencies

```
npm i --legacy-peer-deps
```

### Start the development server

```
npm run dev
```
Open http://localhost:5173 to view the page locally.

### Build for production

```
npm run build
npm run start
```
This builds the project and starts a local server for testing the SSR output.
